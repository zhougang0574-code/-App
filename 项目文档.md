## 一、项目概述

面向 4-8 岁儿童的口算练习 App。自动出题，语音播报题目，语音识别答案，全程离线运行。

**核心功能**
- 三档难度：10 以内加法 / 10 以内加减法 / 20 以内加减法
- 每轮 10 题，语音逐题播报
- 离线语音识别（Vosk），只识别数字，准确率高
- 预录音频播报（讯飞萌萌音色），无需联网，无需系统 TTS
- 答对随机一条鼓励语（高兴语调），答错/超时公布正确答案
- 答错有一次重试机会
- 答题过程屏幕常亮

## 五、语音识别

使用 **Vosk 离线 STT**，无需联网，识别准确率高。

**模型：** `vosk-model-small-cn-0.22`（约 42MB）  
**位置：** `app/src/main/assets/vosk-model-small-cn/`  
**语法约束：** 只识别零一二三四五六七八九十十一…二十，屏蔽无关词汇

首次启动时模型从 assets 解压到 `filesDir`，后续复用缓存。

## 六、游戏流程

```
选难度
  ↓
出 10 道题，逐题：
  ↓
[SPEAKING] 播报题目（数字 + 加/减 + 数字 + 等于多少）
  ↓
[LISTENING] 开始录音，10 秒倒计时
  ├─ 识别到答案 → 校验
  │     ├─ 答对 → 播鼓励语 → 下一题
  │     ├─ 答错（首次）→ 播重试提示 → 继续监听
  │     └─ 答错（已重试）→ 播"答错啦，答案是X" → 下一题
  └─ 超时 → 播"时间到了，答案是X" → 下一题
  ↓
10 题完成 → 结果页（星级评分 + 答题回顾）
```

## 八、构建说明

**首次运行前确认**
1. `app/src/main/assets/audio/` 下有 33 个 MP3（若无，运行生成脚本）
2. `app/src/main/assets/vosk-model-small-cn/` 下有模型文件

**Android Studio 运行**
- 连接 Android 手机，开启开发者模式 + USB 调试
- 点击 Run ▶，首次启动会解压 Vosk 模型（约 10 秒）
- 需授予麦克风权限

**所需权限**
```xml
<uses-permission android:name="android.permission.RECORD_AUDIO" />
```

## 九、后续扩展建议

| 功能 | 说明 |
|------|------|
| 扩大题目范围 | 修改 `Difficulty.kt` 的 `maxNumber`，并补充生成对应数字音频 |
| 新增鼓励语 | 在讯飞脚本中添加条目，重新运行生成 |
| 调整语速/语调 | 修改脚本中 `speed` / `pitch` 参数（0-100） |
| 更换音色 | 替换脚本中 `VOICE_NEUTRAL` / `VOICE_HAPPY` 的音色代码 |
| 增加题数选项 | 在 `DifficultyScreen` 添加选择器，`QuizViewModel.startQuiz` 接收数量参数 |
| 背景音乐 | 使用 `SoundPool` 或独立 `MediaPlayer` 循环播放 |
